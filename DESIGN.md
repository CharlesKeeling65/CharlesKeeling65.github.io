# 视觉与 UI 设计规范 (Claude Vibe x Twilight)

本文档基于 Claude 的温暖人文设计规范（VibeUI），结合 Twilight 主题的丰富华丽感，为本博客提供未来的 UI 和视觉设计准则。

## 核心设计理念 (Core Philosophy)

整体设计应当像一间经过精心布置的书房——温暖、从容、带有书卷气，同时不失数字产品的精致感。我们将使用“仿皮纸”的暖色调画布，取代传统科技产品中冷硬的纯白或蓝灰配色，营造一种更具温度的沉浸式阅读体验。

### 关键特征
- **暖色调纸质背景** (`#f5f4ed`)：取代纯白，提供如高级信笺般的阅读质感。
- **自定义排版结构**：使用带有古典美感的 Serif (衬线体) 作为标题，Sans (无衬线体) 处理 UI 文字。
- **陶土/砖红品牌色** (`#c96442`)：温暖、泥土感，避免科技感过强的刺眼色彩。
- **全暖色系中性色**：所有的灰色和深色都带有微弱的黄褐色或橄榄色基调。
- **光环阴影系统 (Ring-based shadows)**：使用 `0px 0px 0px 1px` 的柔和描边代替生硬的边框，创造微妙的深度。
- **留白与呼吸感**：像阅读一本杂志那样，使用宽松的行高和宽裕的区块间距。

## 色彩系统 (Color Palette)

### 主色与强调色 (Primary)
- **深邃暖黑 (Anthropic Near Black - `#141413`)**：主要文本色及暗黑模式底色。不是死黑，而是带有温度的极深色，更护眼。
- **陶土红 (Terracotta Brand - `#c96442`)**：品牌强调色，用于主按钮、核心高亮。带有泥土气息，平易近人。
- **珊瑚暖红 (Coral Accent - `#d97757`)**：较亮的辅助色，用于暗色背景下的链接和次级强调。

### 背景与表面色 (Surface)
- **皮纸白 (Parchment - `#f5f4ed`)**：核心页面背景，带有微弱黄绿色调的暖白色。
- **象牙白 (Ivory - `#faf9f5`)**：用于卡片和悬浮容器的背景，比底色稍亮，形成微妙层次。
- **暖沙灰 (Warm Sand - `#e8e6dc`)**：按钮和明显的可交互区域背景色。
- **深邃暖灰 (Dark Surface - `#30302e`)**：暗黑模式下的容器背景，偏暖的炭灰色。

### 文本与中性色 (Neutrals & Text)
- **暖木炭灰 (Charcoal Warm - `#4d4c48`)**：浅色表面的次级/按钮文字。
- **橄榄灰 (Olive Gray - `#5e5d59`)**：次要正文文字。
- **石头灰 (Stone Gray - `#87867f`)**：第三级文本、元数据和页脚信息。

### 边框与光环 (Semantic & Accent)
- **奶油边框 (Border Cream - `#f0eee6`)**：几乎隐形的暖色边框，用于最柔和的区块隔离。
- **暖色光环 (Ring Warm - `#d1cfc5`)**：用于按钮悬停或获得焦点时的柔和阴影 (`0px 0px 0px 1px`)。

## 排版系统 (Typography)

* **标题 (Headings)**: 首选 Serif (衬线体，如 `Georgia`, `Playfair Display`)，字体粗细固定在中等 (`500`)，以展现权威感和书籍标题般的庄重感。行高紧凑 (`1.10 - 1.30`)。
* **正文与 UI (Body / UI)**: 使用 Sans (无衬线体，如 `Inter`, `系统默认无衬线`)，提供清晰的阅读和功能引导。正文行高应宽松 (`1.60`)，提供文学阅读体验。
* **胶囊与小标签 (Pills & Labels)**: 使用极小字号的 Sans 字体，增加字母间距 (`0.12px - 0.5px`)，确保小字体的辨识度。

---

## 阴影与深度 (Shadows & Depth)

我们拒绝使用厚重、生硬的 Drop Shadows。深度的表现来自以下几种方式：
1. **容器颜色对比**：通过在皮纸白背景上放置象牙白卡片来体现层级。
2. **描边级阴影 (Ring Shadow)**：`box-shadow: 0 0 0 1px #d1cfc5`，像是一圈微弱的光晕边缘。
3. **耳语阴影 (Whisper Shadow)**：`box-shadow: 0 4px 24px rgba(0,0,0,0.05)`，极度微弱的悬浮感，仅用于需要明显拔高的视觉焦点。

## 组件指导原则 (Components Guidelines)

### 胶囊化标签 (Pills / Badges)
- 用于展示日期、分类、标签等元数据。
- 采用较大的圆角 (`rounded-full` 或 `rounded-2xl`)。
- 背景通常为 `Warm Sand` 或低透明度的 `Terracotta`，搭配小号、大间距的文字，使信息层级分明。

### 模块化卡片 (Cards)
- 边框应该是 `Border Cream`，背景是 `Ivory`。
- 不要使用过于尖锐的直角，采用 `8px` 到 `16px` 的圆角 (`rounded-xl` / `rounded-2xl`)。
- 悬停效果应体现为温和的向上偏移 (`-translate-y-1`) 和光环阴影的加深。

### 贡献度热图 (Heatmap/Contribution Graph)
- 如果引入类似 GitHub 的热图，应避免使用高饱和度的冷绿色。
- 建议的热图色阶：从 `Parchment` (空白/0贡献) -> `Warm Sand` -> `Coral Accent` -> `Terracotta Brand` (高贡献)，以保持全站温暖的色调一致性。
